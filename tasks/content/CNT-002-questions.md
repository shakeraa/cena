## Context

Each concept needs 8-15 questions at varying Bloom levels. Questions are generated by Kimi K2.5 in overnight batches, reviewed by experts, and stored in PostgreSQL. Each question has Hebrew and Arabic text, a correct answer, rubric criteria, and MCQ distractors (for MCQ type). Distractor quality is critical — distractors must represent common student misconceptions, not random wrong answers.

### CNT-002.3: Distractor Quality Validation

**Files to create/modify:**
- `src/llm_acl/batch/distractor_validator.py` — automated distractor quality checks
- `tests/content/test_distractor_quality.py`

**Acceptance:**
- [ ] No distractor is identical to the correct answer
- [ ] No two distractors are identical to each other
- [ ] Each distractor tagged with misconception type: `procedural_slip`, `sign_error`, `order_of_operations`, `conceptual_error`
- [ ] Distractors are plausible (within 2x magnitude of correct answer for numeric)
- [ ] Hebrew distractor text grammatically valid (basic check via language model)
- [ ] Absurd distractors flagged: correct answer is 42, distractor is 42000000

**Test:**
```python
def test_distractors_are_unique():
    questions = load_generated_questions("math-fractions")
    for q in questions:
        if q.type == "mcq":
            all_options = [q.correct_answer] + [d.text for d in q.distractors]
            assert len(all_options) == len(set(all_options))
```

### CNT-002.4: Hebrew+Arabic Bilingual Consistency Check

**Files to create/modify:**
- `src/llm_acl/batch/bilingual_validator.py` — cross-language consistency
- `tests/content/test_bilingual.py`

**Acceptance:**
- [ ] Hebrew and Arabic versions of each question test the same mathematical concept
- [ ] Mathematical expressions identical in both languages (only labels/instructions differ)
- [ ] RTL rendering verified: no LTR leaks in mathematical notation
- [ ] Answer format consistent: if Hebrew expects "x = 5", Arabic expects "x = 5" (not translated digits)

**Test:**
```python
def test_bilingual_math_consistency():
    questions = load_generated_questions("math-addition")
    for q in questions:
        he_math = extract_math_expressions(q.question_he)
        ar_math = extract_math_expressions(q.question_ar)
        assert he_math == ar_math, f"Math mismatch: HE={he_math}, AR={ar_math}"
```

## Rollback Criteria
If generated questions have systematic quality issues:
- Pause batch generation, switch to manual question authoring
- Re-run generation with improved prompts for affected concepts
- Expert review queue prioritized for flagged questions
