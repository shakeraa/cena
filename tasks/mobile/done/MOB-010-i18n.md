## Context

Primary languages: Hebrew (he) and Arabic (ar) — both RTL. English (en) as fallback. Flutter's `intl` package with ARB files. Font switching: Hebrew uses Heebo, Arabic uses Noto Sans Arabic, English uses Inter.

## Subtasks

### MOB-010.1: ARB Files + l10n Setup
- [ ] `lib/l10n/app_he.arb` — 200+ Hebrew strings
- [ ] `lib/l10n/app_ar.arb` — 200+ Arabic strings
- [ ] `lib/l10n/app_en.arb` — English fallback
- [ ] `flutter_localizations` configured in MaterialApp
- [ ] Locale detection: device locale -> app locale, with manual override

### MOB-010.2: Font Switching
- [ ] Hebrew: Heebo (Google Fonts) — designed for Hebrew readability
- [ ] Arabic: Noto Sans Arabic — broad glyph coverage
- [ ] English: Inter — clean, modern
- [ ] Font loaded based on active locale
- [ ] Math expressions always use Latin fonts regardless of locale

### MOB-010.3: RTL Layout
- [ ] `Directionality.of(context)` used for all layout decisions
- [ ] Icons: directional icons mirrored (back arrow, forward arrow)
- [ ] Text alignment: start-aligned (respects directionality)
- [ ] Number rendering: Arabic-Indic numerals optional (default: Western Arabic)

### MOB-010.4: Locale Persistence
- [ ] User's locale choice saved in SharedPreferences
- [ ] Locale change triggers full rebuild via Riverpod
- [ ] Server notified of locale change for LLM prompt language selection
