## Context

SVG diagrams pre-generated by Kimi K2.5 are served from S3/CloudFront. The mobile viewer renders SVGs with interactive hotspots. Challenge cards are SmartyMe-style game cards linked to diagrams.

## Subtasks

### MOB-009.1: SVG Diagram Viewer
- [ ] Render SVG from CDN URL using `flutter_svg`
- [ ] Pinch-to-zoom, pan, double-tap reset
- [ ] Hotspot interaction: tappable regions on diagrams trigger concept info overlays
- [ ] Loading skeleton while CDN fetches

### MOB-009.2: Challenge Card Widget
- [ ] Card format: diagram + question + timer + XP reward
- [ ] Swipe to reveal answer
- [ ] Difficulty indicator (stars 1-5)

### MOB-009.3: CDN Caching Strategy
- [ ] In-memory LRU cache: 50 most recent diagrams
- [ ] Disk cache: `cached_network_image` with 7-day TTL
- [ ] Prefetch: load next 3 diagrams during session idle time
- [ ] Offline: show cached diagrams, placeholder for uncached
