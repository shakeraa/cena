# STEM Visualization & Scientific Diagram Tools - Competitive Intelligence Report

**Research Date:** February 2026  
**Analyst:** Competitive Intelligence Team  
**Competitors Analyzed:** FigureLabs, GPAI, Nibble  

---

## Executive Summary

This report analyzes three emerging competitors in the STEM visualization and scientific diagram space. **FigureLabs** leads in AI-powered scientific illustration with text-to-figure generation and publication-ready exports. **GPAI** focuses on STEM homework solving with integrated diagram visualization. **Nibble** takes a microlearning approach with interactive content across 20+ topics including STEM subjects.

**Key Finding:** These tools represent a **feature category CENA should integrate** rather than compete against directly. Their diagram generation capabilities could enhance CENA's existing interactive educational content.

### Visualization Deep-Dive

**AI Generation Capabilities:**
- **Text-to-Figure:** Generate complex scientific diagrams from text descriptions in seconds
- **Sketch-to-Figure:** Transform hand-drawn sketches into professional scientific images
- **Figure Refiner:** Enhance existing figures via resolution enhancement, color correction, noise reduction
- **Image Vectorization:** Convert raster images to scalable SVG for easy editing

**Export Options:**
- SVG (vector, fully editable)
- PNG (high-res, publication-ready)
- JPG (optimized for web)
- **PPTX (editable PowerPoint slides)** - Key differentiator

**AI Models Used:**
- Nano Banana Pro (Google Gemini 3 Pro Image)
- Nano Banana 2
- GPT Image 1.5
- Flux Pro 1.1, SeeDream 4.5, Sora, Flux.2 Max (higher tiers)

## Strategic Recommendations for CENA

### 1. Near-Term (0-3 months): Partner for Visualization

**Action:** Explore API integration with FigureLabs or similar tools

**Rationale:**
- FigureLabs has solved the "publication-ready diagram" problem
- CENA provides the learning context and pedagogy
- Faster time-to-market than building internally

**Implementation:**
- Embed diagram generation within lesson content creation
- Allow students to generate custom diagrams for their notes
- Export to CENA's existing interactive format

### 2. Medium-Term (3-6 months): Build STEM Solver

**Action:** Develop CENA-native STEM problem-solving with visualization

**Rationale:**
- GPAI's triple cross-check approach addresses AI accuracy concerns
- Differentiates CENA from generic AI tutors
- High student demand for homework help

**Implementation:**
- Photo upload for problem capture
- Step-by-step solution generation
- Auto-generated diagrams for visual learners
- Integration with CENA's spaced repetition system

### 3. Long-Term (6-12 months): 3D Visualization

**Action:** Evaluate 3D/CAD visualization capabilities

**Rationale:**
- None of the competitors offer true 3D modeling
- Engineering and architecture students need 3D visualization
- Significant differentiation opportunity

**Implementation:**
- Partner with CAD visualization libraries
- Integrate with CENA's interactive lesson format
- Enable manipulation of 3D models within lessons

### 4. Messaging Opportunities

**Positioning CENA vs. These Tools:**

| Competitor Weakness | CENA Strength |
|---------------------|---------------|
| FigureLabs: No learning platform | "Learn with AI-generated visuals, not just create them" |
| GPAI: No structured curriculum | "Master STEM with guided learning paths, not just answers" |
| Nibble: No AI generation | "Interactive AI-generated content, not static lessons" |
