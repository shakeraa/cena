# CENA UI/UX Design Resources Report
## Figma Designs & Flutter Templates for 14-18 Age Group (High School)

---

## Executive Summary

This report curates Figma design resources and Flutter UI kits that match 2026 design trends and are specifically suitable for the 14-18 high school demographic. The focus is on **mature, minimalist, professional aesthetics** with dark mode support, glassmorphism effects, and clean typography.

**Target Audience Profile (Ages 14-18):**
- Abstract thinkers with longer attention spans (20-30 minutes)
- Goal-oriented and self-directed learners
- Dislike "childish" or patronizing design
- Prefer dark mode (82.7% adoption rate)
- Expect premium, app-store-quality experiences
- Value social features and peer comparison

## Part 3: Glassmorphism/Liquid Glass Flutter Packages

### 3.1 glassmorphism (Pub.Dev)

**Link:** https://pub.dev/packages/glassmorphism

**Features:**
- ✅ Easy glassmorphic containers
- ✅ Highly customizable
- ✅ Cross-platform (iOS, Android, Web, Desktop)
- ✅ 522 GitHub stars

**Usage:**
```dart
GlassmorphicContainer(
  width: 350,
  height: 750,
  borderRadius: 20,
  blur: 20,
  alignment: Alignment.bottomCenter,
  border: 2,
  linearGradient: LinearGradient(
    begin: Alignment.topLeft,
    end: Alignment.bottomRight,
    colors: [
      Color(0xFFffffff).withOpacity(0.1),
      Color(0xFFFFFFFF).withOpacity(0.05),
    ],
  ),
  borderGradient: LinearGradient(
    begin: Alignment.topLeft,
    end: Alignment.bottomRight,
    colors: [
      Color(0xFFffffff).withOpacity(0.5),
      Color(0xFFFFFFFF).withOpacity(0.5),
    ],
  ),
  child: YourWidget(),
)
```

## Part 5: Design Recommendations for CENA (14-18)

### 5.1 Visual Style Guidelines

**Color Palette:**
```
Primary:    #3B82F6 (Blue - trust, focus)
Secondary:  #8B5CF6 (Purple - wisdom, premium)
Success:    #10B981 (Green - growth, correct)
Warning:    #F59E0B (Orange - energy, streaks)
Danger:     #EF4444 (Red - errors, urgency)

Dark Background:  #0F172A (Slate 900)
Dark Surface:     #1E293B (Slate 800)
Light Background: #F8FAFC (Slate 50)
Light Surface:    #FFFFFF (White)
```

**Typography:**
- Primary: Inter or SF Pro (modern, clean)
- Headings: 24-32px, Bold
- Body: 16px, Regular
- Caption: 14px, Medium
- Use dynamic type for accessibility

**Spacing:**
- Base unit: 8px
- Small: 8px
- Medium: 16px
- Large: 24px
- XLarge: 32px

---

### 5.2 Key UI Patterns for 14-18

**Dashboard (Bento Grid):**
```
┌─────────────────────────────────────┐
│  HEADER: Avatar | Streak | Settings │
├─────────────┬───────────┬───────────┤
│  PROGRESS   │  DAILY    │  BADGES   │
│  RING       │  CHALLENGE│  SHELF    │
├─────────────┴───────────┴───────────┤
│      CONTINUE LEARNING (Hero)       │
├─────────────┬───────────┬───────────┤
│  SKILL TREE │  LEADER-  │  ANALYTICS│
│             │  BOARD    │           │
└─────────────┴───────────┴───────────┘
```

**Quiz Interface:**
- Clean, minimal design
- Large touch targets (60pt+)
- Clear visual hierarchy
- Immediate feedback
- Progress indicator

**Dark Mode Default:**
- True black (#000000) for OLED
- Charcoal surfaces (#1E293B)
- Bright accents only where needed
- Reduced visual noise

---

### 5.3 Animation Guidelines

**Timing:**
- Micro-interactions: 150-200ms
- Screen transitions: 300-400ms
- Celebrations: 500-800ms
- Progress fills: 1000ms

**Easing:**
- Standard: ease-in-out
- Enter: ease-out
- Exit: ease-in
- Bounce: for celebrations

**Effects:**
- Subtle scale on button press (0.95x)
- Smooth opacity transitions
- Gentle spring animations
- Avoid excessive motion

---

### 5.4 Accessibility Requirements

- Minimum contrast: 4.5:1 for text
- Touch targets: 44x44pt minimum
- Dynamic type support
- Reduced motion option
- Screen reader support
- Color not the only indicator
