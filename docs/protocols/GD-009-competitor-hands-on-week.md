## Per-product report template

1. **First impression** (30 sec): what happens on first open?
2. **Onboarding**: how many steps? How long? What's asked?
3. **Core loop**: describe the session-as-game-loop
4. **Gamification**: streaks? XP? Leaderboards? Dark patterns?
5. **Pedagogy**: step-level feedback? Misconception awareness? Scaffolding?
6. **RTL/Arabic**: any support? Quality?
7. **Mobile UX**: responsive? Touch-optimized? Offline?
8. **Steal this**: one feature Cena should adopt
9. **Avoid this**: one anti-pattern Cena must not copy
