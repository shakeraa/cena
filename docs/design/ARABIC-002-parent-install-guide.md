# Arabic Parent Install Guide — Cena (ARABIC-002)

> **Purpose**: PDF guide for Arabic-speaking parents to install Cena PWA
> **Languages**: Arabic (primary), with screenshot labels in Arabic
> **Distribution**: QR code in pilot school welcome packets (Nazareth, Umm al-Fahm, Rahat)

## Content outline

### Page 1: Welcome
- Cena logo + "مرحباً بكم في سينا" (Welcome to Cena)
- What Cena does: adaptive math + physics tutoring for Bagrut
- No app store needed — works in the browser

### Page 2: Installation steps (iPhone)
1. Open Safari (screenshot)
2. Navigate to cena.app (screenshot)
3. Tap share icon → "Add to Home Screen" (screenshot)
4. Tap "Add" (screenshot)
5. Open Cena from home screen (screenshot)

### Page 3: Installation steps (Android)
1. Open Chrome (screenshot)
2. Navigate to cena.app (screenshot)
3. Tap "Install" banner or ⋮ → "Install app" (screenshot)
4. Tap "Install" (screenshot)
5. Open Cena from home screen (screenshot)

### Page 4: First login
1. Student taps "Create account" (screenshot)
2. Parent enters email for verification (COPPA/parental consent)
3. Student picks curriculum track (GD-008 Bagrut Physics 036)
4. Student enters classroom join code (from teacher)

### Page 5: Privacy & safety
- "Your child's data is protected" — plain-language privacy summary
- No personal data shared with other students
- Parent can request data export or deletion at any time
- Contact: parent-support@cena.app
